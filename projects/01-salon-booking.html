Salon Appointment Booking Automation

A fully automated booking system that eliminates manual phone calls and back-and-forth messages. From form submission to calendar event, confirmation, and logging, everything happens without human intervention.

Typeform / Tally Google Calendar Twilio SMS Gmail Google Sheets n8n

What It Does

This workflow removes the back-and-forth that typically bogs down salon booking. When a client fills out the online form, the workflow takes over entirely. No receptionist needed, no double-bookings, no missed confirmations.

It validates the submission, checks Google Calendar for conflicts, creates the event if the slot is available, and fires off both an SMS and email confirmation to the client, all within seconds. If the slot is already taken, the client receives an automated notice to pick a different time.

Every booking is logged to Google Sheets automatically, giving the salon owner a clean audit trail and easy reporting without lifting a finger.

How It Works

  1. Trigger Client submits the booking form via Typeform or Tally. The webhook fires instantly, starting the workflow.
  2. Validate Required fields (name, email, phone, service, requested date/time) are checked. Missing fields return a 400 response immediately. No bad data enters the system.
  3. Normalise A Set node whitelists only the fields the workflow needs, stripping everything else for a clean, lean payload.
  4. Check Availability Google Calendar is queried for conflicts in the requested 1-hour slot against the salon's calendar ID.
  5. Create Booking If the slot is free, a calendar event is created automatically with all client details attached.
  6. Confirm Twilio sends an SMS confirmation and Gmail sends a branded email, both within seconds of the booking being made.
  7. Log The full booking record is appended to Google Sheets for reporting, follow-up, and audit trail.

Business Outcome

Eliminates manual booking calls and removes double-booking risk entirely. Saves approximately 4 hours per week in admin time and reduces no-shows through instant, automated confirmations sent the moment a booking is made.
